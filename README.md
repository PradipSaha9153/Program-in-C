# Program-in-C

This repository contains all my C programming practice work completed during my BCA (Bachelor of Computer Applications). It includes basic to intermediate level programs that help in understanding core concepts of the C language.

## 📌 About

The purpose of this repository is to:

* Practice and improve C programming skills
* Build a strong foundation in programming concepts
* Maintain a structured collection of all practice programs

## 📂 Contents

The repository includes programs related to:

* Basic syntax and structure
* Input/Output operations
* Conditional statements (if, switch)
* Loops (for, while, do-while)
* Functions
* Arrays and strings
* Pointers
* Structures and unions
* File handling (if included)

## 🚀 How to Run

1. Install any C compiler (like GCC)
2. Clone this repository:

   ```bash
   git clone https://github.com/your-username/Program-in-C.git
   ```
3. Navigate to the folder and compile any program:

   ```bash
   gcc filename.c -o output
   ./output
   ```
